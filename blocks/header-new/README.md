# Header New Block

## Overview
Extended header component with authentication features and user account dropdown functionality.

## Purpose
Provides secondary navigation bar with user authentication state management and account-related actions.

## Configuration
No configuration options required.

## Integration
- Renders automatically as part of the main header
- Integrates with authentication state
- Works with header navigation system

## Behavior
- Displays user account status
- Provides login/logout functionality
- Shows user-specific navigation options

## Error Handling
- Gracefully handles missing authentication data
- Falls back to guest state when user not logged in
