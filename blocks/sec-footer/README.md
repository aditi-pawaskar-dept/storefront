# Secondary Footer Block

## Overview
Secondary footer component with additional site information and links.

## Purpose
Provides supplementary footer content, social links, and additional navigation.

## Configuration
No configuration options required.

## Features
- **Social Links**: Icons for social media platforms
- **App Downloads**: Links to mobile app stores
- **Additional Navigation**: Secondary site links
- **Contact Information**: Support and contact details

## Integration
- Renders below main footer
- Integrates with site-wide footer system
- Uses icon system for social media

## Behavior
- Displays consistently across all pages
- Links open in appropriate contexts
- Responsive layout for different screen sizes

## Error Handling
- Handles missing content gracefully
- Provides fallback for missing icons
- Maintains layout integrity
